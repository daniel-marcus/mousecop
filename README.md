# 🐭 Mousecop

A macOS training tool for keyboard-only navigation. Every time you touch the mouse, your screen turns red. Mousecop is watching!

![](screenshot.png)

## Development

Using [just](https://github.com/casey/just) and [watchexec](https://watchexec.github.io/)

```sh
just develop
```

## Build App

```sh
just build
```
